# MVP Scope

## Roles
### Public
- View events
- Register as usher or volunteer

### Usher
- Login via Google
- View upcoming assignments

### Core Leader (Admin)
- Manage events
- View registrations
- Assign ushers
- View usher database
- Birthday-by-month list

## Must-Have Features
- Authentication (Google OAuth)
- Role-based access
- Event CRUD
- Registration system
- Admin dashboard

## Deferred Features
- Push notifications
- Attendance tracking
- Native apps
